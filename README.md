# Demo Assignment 1

Personal portfolio website built with Express and Handlebars.

## Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the app:
   ```bash
   npm start
   ```
3. Open http://localhost:3000

## Project Structure

- `app.js` - Express application setup
- `bin/www` - server launcher
- `routes/index.js` - page routes
- `views/` - Handlebars templates
- `public/stylesheets/style.css` - site styling
